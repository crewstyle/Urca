Getting started

Urca's buttons allow for a ton of flexibility when building out user interfaces.
There are two ways to create a button:

  • The <a> tag if the button is a link to another page
  • The <button> tag if the button performs an action that changes something on the current page


<a href="#" class="btn">Button</a>
<button class="btn">Button</button>
