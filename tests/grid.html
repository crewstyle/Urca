Urca › Grid

Getting started

Urca's grid includes a responsive, mobile-first 12-column fluid grid system. A container is provided as a wrapper to set a max width, which can be easily overrided. The syntax is simple and semantic.
It uses new flex CSS attribute instead of inline-block display or float elements.

Options

Rows and cols

You can easily build your web applications with rows and columns. The main structure of your application can be set in a single row thanks to the .row or .rows CSS classes.
Urca's grid provide two ways to set your column's width in addition to the .col or .cols CSS classes:

  • define width specifically for each one
  • let the bowser defining it for you

Here is an example with specified widths:

<!-- You specify a width for each column -->
<section class="row">
    <article class="col four">item 1</article>
    <article class="col four">item 2</article>
    <article class="col four">item 3</article>
    <article class="col four">item 4</article>
</section>
