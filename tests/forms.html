Urca › Forms

Forms have always been tricky.

Getting started

Creating a form with Urca is designed to be easy but extremely flexible. Forms are built with a combination of standard form elements, as well as grid rows and columns.

A simple example 

<form method="post" action="" autocomplete="off">
    <fieldset>
        <legend>A simple example</legend>
        <label>Input text</label>
        <input type="text" placeholder="Text" />
    </fieldset>
</form>

Group and labels

Working with forms is quite simple. If you need some more display options, you can use .form-group and .form-labels CSS classes to make your display even better.

Examples 

<form method="post" action="" autocomplete="off">
    <fieldset>
        <legend>Examples</legend>

        <div class="form-group">
            <label>Input text</label>
            <input type="text" placeholder="Text" />
        </div>

        <div class="form-labels">
            <label class="label-block">Checkboxes in a <code>.form-labels</code></label>
            <label><input type="checkbox" /> Checkbox #1</label>
            <label><input type="checkbox" /> Checkbox #2</label>
            <label><input type="checkbox" /> Checkbox #3</label>
        </div>
    </fieldset>
</form>

Form stacked

To create a stacked form, add the form-stacked classname to a <form> element.

Please, sign in! 

<form method="post" action="" autocomplete="off" class="form-stacked">
    <fieldset>
        <legend>Please, sign in!</legend>

        <div class="form-group">
            <label>Email</label>
            <input type="email" placeholder="Email" />
        </div>

        <div class="form-group">
            <label>Password</label>
            <input type="password" placeholder="Password" />
        </div>

        <div class="form-group">
            <label for="remember" class="form-label"><input type="checkbox" id="remember" /> Remember me</label>
        </div>

        <div class="form-submit">
            <button type="submit" class="btn primary">Sign in</button>
            <button type="reset" class="btn link">Cancel</button>
        </div>
    </fieldset>
</form>

Form aligned

To create an aligned form, add the form-aligned classname to a <form> element instead of form-stacked classname. In an aligned form, the labels are right-aligned against the form input controls.

Fields

Text inputs

These input types create a text field: color, date, datetime, datetime-local, email, month, password, search, tel, text, time, url and week.

A simple example 

<label>Password</label>
<input type="password" placeholder="Your password here" value="password" />

Numbers

These input types create a text field: number and range.

A simple example

Textareas

These input types create a text field: textarea.

A simple example 

<label>Textarea</label>
<textarea placeholder="7"></textarea>

Selects

These input types create a text field: select.

A simple example
